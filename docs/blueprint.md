# **App Name**: CourseFlow

## Core Features:

- Course Browsing and Enrollment: Browse available courses, filter by categories and difficulty, and enroll in selected courses.
- Video and Text Lesson Playback: Watch videos and read course notes within the platform, with a split layout for video/text and notes.
- Progress Tracking: Automatically save and display progress after each lesson. Progress data updates in real time.
- Note-Taking: Allow the users to create, save, and manage lesson-specific notes and bookmarks. Notes persist in Cloud SQL.
- Admin Panel: Administer user accounts, moderate content and instructor access
- Personalized Recommendations: AI-powered course recommendation tool system to suggest relevant courses based on user activity, enrollment, and preferences.

## Style Guidelines:

- Primary color: Indigo (#4F46E5) for a professional and trustworthy feel.
- Background color: Light gray (#F9FAFB), offering a clean and modern look with minimal distraction.
- Accent color: Teal (#008080) to highlight key elements and calls to action.
- Body and headline font: 'Inter' (sans-serif) providing a modern, machined, neutral look; suitable for headlines and body text
- Code font: 'Source Code Pro' for displaying code snippets.
- Use a minimalistic layout focused on usability, responsive design, soft shadows, rounded corners (2xl), and subtle gradients.
- Incorporate subtle animations using Framer Motion for a smooth, interactive user experience.
